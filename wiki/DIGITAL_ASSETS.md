# 🗺️ 潮汐投研 (TideTrading) 数字资产与域名解析规范

本文件记录了 潮汐投研 (TideTrading) 项目相关的域名资产分配以及阿里云 DNS 解析配置规范。

---

## 1. 🏷️ 品牌与多语言本地化配置

*   **项目默认名称**：潮汐投研 (英文名: TideTrading)
*   **默认语言**：中文 (Chinese)
*   **多语言支持**：英语 (English)
*   **正式环境生产域名**：`https://tide.red/`

---

## 2. 🌐 域名资产分配与阿里云 DNS 解析

为了实现潮汐投研多租户的稳定隔离与测试灰度访问，配置了以下域名资产规范：

*   **正式生产域名 (Production Domain)**：`tide.red`
    *   **用途**：潮汐投研主要的公网访问入口，承载 Web 端交易控制台、公共 API 网关及对外 MCP 工具注册。
    *   **托管解析**：在阿里云解析 DNS 控制台中配置 `A` 记录指向生产环境固定公网 IP。
*   **测试域名 (Testing Domain)**：`*.bbr` (如 `test.bbr` 或 `dev.bbr`)
    *   **用途**：测试环境（灰度 API 调试端口、新版本发布测试）的公网穿透访问入口。
    *   **解析记录**：阿里云 DNS 控制台中同时配置了相应的 `TXT` 解析校验记录，用于完成 SSL 证书的所有权检验。
